# Tarefa de aula — API autenticada de Produtos

Atualize o projeto Laravel desenvolvido em aula para permitir que as operações de **cadastrar, atualizar e excluir produtos** sejam realizadas por meio da API, utilizando autenticação com Laravel Sanctum.

O endpoint de listagem de produtos foi desenvolvido junto com o professor em aula. A partir dele, cada aluno deverá implementar as demais operações da API.

---

## O que deve ser implementado

O aluno deve criar os endpoints:

```http
POST /api/products
```

```http
PUT /api/products/{id}
```

```http
DELETE /api/products/{id}
```

Essas rotas devem estar protegidas por autenticação com Sanctum utilizando:

```php
auth:sanctum
```

---

## Requisitos

A API deve permitir:

- cadastrar um novo produto;
- atualizar os dados de um produto existente;
- excluir um produto;
- validar os dados recebidos na requisição;
- retornar respostas em JSON;
- exigir token de autenticação para acessar as rotas protegidas.

---

## Testes

Os endpoints devem ser testados utilizando uma ferramenta como:

- Thunder Client;
- REST Client no VS Code;
- Postman;
- Insomnia.
