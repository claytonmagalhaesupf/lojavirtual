# Aula — Seeders, Factories e Paginação no Laravel

# Parte 1 — Factories e Seeders

## Criar Factory

```bash
php artisan make:factory ProductFactory --model=Product
```

---

## Arquivo

```txt
database/factories/ProductFactory.php
```

---

## ProductFactory

```php
<?php

namespace Database\Factories;

use App\Models\Product;
use App\Models\Type;
use Illuminate\Database\Eloquent\Factories\Factory;

class ProductFactory extends Factory
{
    protected $model = Product::class;

    public function definition(): array
    {
        return [
            'name' => fake()->words(3, true),
            'description' => fake()->sentence(10),
            'quantity' => fake()->numberBetween(1, 200),
            'price' => fake()->randomFloat(2, 10, 1000),
            'image' => null,
            'type_id' => Type::inRandomOrder()->first()?->id ?? Type::factory(),
        ];
    }
}
```

---

## Criar Seeder

```bash
php artisan make:seeder ProductSeeder
```

---

## Arquivo

```txt
database/seeders/ProductSeeder.php
```

---

## ProductSeeder

```php
<?php

namespace Database\Seeders;

use App\Models\Product;
use Illuminate\Database\Seeder;

class ProductSeeder extends Seeder
{
    public function run(): void
    {
        Product::factory()->count(500)->create();
    }
}
```

---

## Registrar Seeder

Arquivo:

```txt
database/seeders/DatabaseSeeder.php
```

```php
<?php

namespace Database\Seeders;

use Illuminate\Database\Seeder;

class DatabaseSeeder extends Seeder
{
    public function run(): void
    {
        $this->call([
            ProductSeeder::class,
        ]);
    }
}
```

---

## Executar Seeders

```bash
php artisan db:seed
```

Ou:

```bash
php artisan migrate:fresh --seed
```

# Parte 2 — Paginação

## Antes

```php
$products = Product::with('type')->get();
```

---

## Depois

```php
$products = Product::with('type')->paginate(10);
```

---

## Controller completo

```php
public function index()
{
    $products = Product::with('type')->paginate(10);

    return view('products.index', compact('products'));
}
```

---

## Adicionar paginação no Blade

Adicionar abaixo da tabela:

```blade
<div class="mt-6">
    {{ $products->links() }}
</div>
```

---

# Como funciona o links()

O método:

```blade
{{ $products->links() }}
```

gera automaticamente:
- links de navegação;
- botão anterior;
- botão próximo;
- numeração das páginas.

Exemplo:
- Página 1
- Página 2
- Página 3

---

# Como funciona internamente

Quando usamos:

```php
paginate(10)
```

o Laravel retorna um objeto especial de paginação.

Esse objeto possui métodos como:

```php
$products->currentPage()
$products->lastPage()
$products->total()
$products->perPage()
```

O método:

```blade
links()
```

utiliza essas informações para montar automaticamente a navegação HTML.

# SQL gerado conceitualmente

```sql
SELECT * FROM products LIMIT 10 OFFSET 0;
```

# Tradução da paginação para PT-BR

Por padrão, a paginação aparece em inglês.

Exemplo:
- Previous
- Next

---

## Instalar traduções

```bash
composer require laravel-lang/common --dev
```

---

## Publicar idiomas

```bash
php artisan lang:add pt_BR
```

---

## Configurar locale

Arquivo:

```txt
config/app.php
```

Alterar: 

```php
'locale' => 'pt_BR',
```

---

# Resultado esperado

Agora a paginação aparecerá em português.

Exemplo:
- Anterior
- Próximo
